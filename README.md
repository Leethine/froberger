# About Froberger

The Froberger project aims to create an online version of LilyPond sheet music editor that is similar to Overleaf and Frescobaldi. It aims to re-create a subset of features of [Frescobaldi](https://www.frescobaldi.org/).

Johann Jakob Froberger (1616 - 1667) was one of the most important composers of the Baroque era. Froberger was the most prominent student of Girolamo Frescobaldi.
